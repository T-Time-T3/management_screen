# management_screen
運営管理画面のレポジトリです．

## DEMO
- 勉強会の登録(主催者)
- 各勉強会の参加/不参加の回答受付
- 勉強会参加者を集計
- LINE Botによる投票の促し，その他連絡（個人）
- Discord Botによる勉強会のリマインド，投票の促し(全体)
